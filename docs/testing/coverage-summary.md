# Coverage Summary

## Human-readable summary (default)

```bash
scripts/coverage/generate-coverage-summary.sh --xcresult <path-to.xcresult>
```

## Machine-readable report (optional)

```bash
scripts/coverage/generate-coverage-summary.sh --xcresult <path-to.xcresult> --machine
```

Or specify an output path:

```bash
scripts/coverage/generate-coverage-summary.sh --xcresult <path-to.xcresult> --machine coverage-report.json
```
